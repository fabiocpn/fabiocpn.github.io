Integrative Analysis

An integration of data across the capstone projects to build a model taking QTLs as inputs and providing both phenotype predictions as well as functional modules involved.

Integrative Analysis for each brain phenotype

  1. Integrative model parameters for all phenotypes:

    • Logistic Regression-Genotype:

      • LR-gen xlsx

    • Logistic Regression-Transcriptome:

      • LR-trans xlsx

    • Conditional Restricted Boltzmann Machine:

      • cRBM xlsx

    • Deep Structured Phenotype Network with Imputation:

      • DSPN-Imput xlsx

    • Modular Deep Structured Phenotype Network (uses WGCNA modules shared in this section):

      • DSPN-mod xlsx

    • Full Deep Structured Phenotype Network:

      • DSPN-full xlsx

  2. Multi-level functional enrichment analysis (DSPN-mod) and Weighted Gene Co-Expression Analysis (WGCNA) modules:

    • Pathway and cell-type enrichment scores xlsx

    • WGCNA modules, Ensembl IDs xlsx

    • WGCNA modules, HGNC IDs xlsx

  3. Transcription Factor - Target Gene - Enhancer linkages:

    • Reference linkages based on TF binding site analysis (before Elastic Net Regression):
      • Reference TF - target gene - enhancer links csv
    • Final linkages in Gene Regulatory Network (with Elastic Net regression weight cutoff = 0.1):
      • Final TF - target gene - enhancer links csv

    • Mean square error for each gene from the Elastic Net Regression (only including genes with Elastic net predictions):
      • Mean Square Errors csv

  4. HiC-derived Enhancer - Gene linkages:

    • HiC Enhancer - Gene linkages csv

  5. Schizophrenia-associated genes:

    • SCZ High-confidence Gene List csv
